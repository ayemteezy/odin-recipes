## About The Project

[![Odin Recipes Screen Shot][product-screenshot]](images/full-design.png)

This project is a multi-page website built to demonstrate foundational web development structures using basic HTML. The goal of this first project from The Odin Project's Foundations course is to learn how to structure data, organize directories, and connect pages without relying on advanced styling rules.

Key elements practiced in this project:

- Writing clean, structured text elements using HTML paragraph tags and header tiers
- Configuring relative link pathways to connect a central homepage with local document assets
- Embedding local media and scaling visual sizing variables directly within document tags
- Creating structured, itemized steps using ordered and unordered nesting variations

## Getting Started

To view or test this project locally, you only need a web browser—no complex setup required.

### Prerequisites

You only need a modern web browser (like Chrome, Firefox, Safari, or Edge) and a text editor if you want to inspect or modify the code.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ayemteezy/odin-recipes.git
   ```
2. Open the directory
   ```sh
   cd odin-recipes
   ```
3. Open `index.html` in your web browser to view the page locally.

## Usage

This project showcases a clean, functional recipe repository featuring:

- **Main Directory**: A centralized home landing page containing links to the entire recipe collection
- **Individual Recipe Pages**: Standalone document files dedicated to specific dishes
- **Visual Previews**: Properly rendered culinary imagery for every featured recipe
- **Ingredient Lists**: Unordered overview groups showcasing all required components
- **Preparation Layouts**: Step-by-step ordered lists outlining precise cooking workflows

Feel free to reuse this file framework pattern for other basic multi-page HTML text mockups or file routing assignments.
